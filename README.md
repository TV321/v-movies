# v-movies
Simple movie app
### Made with:
- Vue.js
- Vuetify
- Axios
- The Movie Database: https://www.themoviedb.org/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
